Welcome to the assignment!

The assignment is consisting from the following tasks:

Task 1: Create a web page to display sales data from a CSV file

  • Use HTML, CSS, and JavaScript to create a web page that displays sales data from the provided CSV file.
  • The web page should include a table to present the data in a tabular format.
  • Parse the CSV file using Python and retrieve the necessary data to populate the table.
  • Apply appropriate styling using CSS to enhance the visual appearance of the table and the web page.
  • Implement JavaScript functionality to allow sorting the table by different columns (e.g., by date, product, or sales volume) when clicking on the table headers.
  • Add a search input field that filters the table rows based on user input. The filtering should be performed dynamically as the user types.
  • Make sure the web page is responsive and displays well on different screen sizes.

Task 2: Create a page with line chart to show "Cumulative Total Sales per Month".

Task 3: Create a page with bar chart to show "Monthly Total Sales for each Territory" (sample look https://slidebazaar.com/items/sample-bar-chart-powerpoint-template/) and implement a button to change the chart to show "Yearly" values for each Territory, button should work with Javascript without loading any other page).

Task 4: Create a page with a 2 pie charts to show "Total Sales by DEAL_SIZE" and "Total Discount/Surcharge by DEAL_SIZE".

Task 5: Create a main page with links to previous 4 pages.
